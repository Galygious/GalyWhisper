# Contributing to GalyWhisper

Thank you for your interest in contributing to GalyWhisper! This document provides guidelines and instructions for contributing.

## Code of Conduct

Please be respectful and considerate of others when contributing to this project.

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature or bugfix
3. Make your changes
4. Submit a pull request

## Development Setup

1. Ensure you have the required dependencies installed:
   - Go 1.21 or later
   - Whisper.cpp
   - PortAudio

2. Clone your fork:
```bash
git clone https://github.com/yourusername/GalyWhisper.git
cd GalyWhisper
```

3. Add the original repository as upstream:
```bash
git remote add upstream https://github.com/original-owner/GalyWhisper.git
```

## Pull Request Process

1. Update the README.md with details of changes if needed
2. Update the documentation if you're changing functionality
3. The PR will be merged once you have the sign-off of at least one other developer

## Style Guide

- Follow Go's official [Code Review Comments](https://github.com/golang/go/wiki/CodeReviewComments)
- Use meaningful commit messages
- Write tests for new functionality
- Keep the code clean and well-documented

## Questions?

Feel free to open an issue if you have any questions about contributing.
